# 演示剧本

## 模式说明

- 白色圆点 = LLM 智能整理模式（默认）
- 蓝色圆点 = 纯转录模式（Ctrl+Alt+B 切换）

---

## LLM 模式演示（白点）

### 场景一：写邮件

说：帮我写一封请假邮件，昨天就有点发烧今天还没好，跟张经理请个病假

期望输出：
```
Subject: 病假申请

张经理您好，

我从昨天开始发烧，今天仍未好转，特申请病假一天。相关工作我会尽量远程处理。

祝好，
[您的名字]
```

### 场景二：会议记录转结构化总结

说：今天例会讨论了三个事情。第一，新版本下周五上线，王磊负责后端接口，李娜负责前端页面。第二，上次的 bug 修复需要重新测试，张伟来负责。第三，下周要准备季度汇报的 PPT，我来写。

期望输出：
```
1. 新版本下周五上线：王磊（后端）、李娜（前端）
2. Bug 修复重新测试：张伟负责
3. 季度汇报 PPT：我来写
```

### 场景三：长段口述整理

说一段较长的内容，LLM 会去除口语冗余，整理为通顺文章。

---

## 纯转录模式演示（蓝点，Ctrl+Alt+B 切换）

### 场景四：快速输入

切到蓝点模式，说：今天下午三点开会，请大家准时参加。

松手立即输出：今天下午三点开会，请大家准时参加。

特点：不经过 LLM，速度快，适合日常快速输入。
